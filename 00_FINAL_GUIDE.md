### 特性 3：自动 Git 同步

**同步流程**：

```
你在工作 PC 学习:
  ↓
系统自动记录学习进度
  ↓
auto-syncer 检测文件变化
  ↓
自动 commit: "Update [文件名]: 学习进度更新"
  ↓
自动 push 到 GitHub
  ↓

你在私人 PC 或手机:
  ↓
auto-syncer 定时 pull（每 5 分钟）
  ↓
获取最新学习进度
  ↓
看到最新的学习记录
```

**冲突处理**：

```python
if 检测到冲突:
    if is_progress_file():
        # 进度文件：合并数据
        return merge_progress_data()
    elif is_question_file():
        # 题库文件：保留最新时间戳
        return keep_newer_version()
    else:
        # 其他文件：标记冲突
        return mark_for_manual_resolution()
```

## 🔧 配置和调优

### 快速配置

首次使用需要配置：

```yaml
# 在 config/settings.md 中配置

automation:
  autonomous_mode: true      # 启用完全自主模式
  auto_commit: true           # 自动 git commit
  auto_push: true             # 自动 git push
  auto_pull: true             # 自动 git pull

learning:
  daily_hours: 4              # 每日学习时间（小时）
  efficiency_threshold: 0.7    # 效率阈值
  review_interval_days: 7        # 复习间隔（天）

sync:
  pull_interval: 300           # pull 间隔（5 分钟）
  debounce_seconds: 1         # 防抖间隔（1 秒）
```

## 📞 需要帮助？

### 常见问题

**Q1：如何开始使用？**
A：打开 Claude Code，说"今天学什么"或"开始学习"，系统会自动开始。

**Q2：我可以干预学习内容吗？**
A：可以！有两种方式：
   - 明确指定："今天学 LangGraph"
   - 调整计划："增加更多练习题"

**Q3：数据会丢失吗？**
A：不会！数据存储在你的 Git 仓库中：
   - Mac mini 24/7 自动同步
   - 你可以在任何设备访问
   - 完整的版本历史

**Q4：系统会学错内容吗？**
A：系统会基于以下做决策：
   - 你的目标职位
   - 当前进度
   - 薄弱点分析
   - 市场趋势
   - 学习效率

**Q5：如何查看系统在做什么？**
A：系统会在每个步骤开始和结束时告知你，例如：
   - "📊 分析当前状态..."
   - "📋 智能决策：今日建议..."
   - "🚀 开始自主执行..."
   - "✅ 完成！"

**Q6：如何修改自动化的规则？**
A：可以！两种方式：
   - 修改配置文件：config/settings.md
   - 明确指定："调整学习强度：高/中/低"
